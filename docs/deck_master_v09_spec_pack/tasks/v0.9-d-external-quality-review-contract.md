# Task D — External Quality Review Contract

## Goal
Let external Agents perform semantic/visual/client-readiness review and import findings into Deck Master Quality Gate.

## Scope
Implement:
- `prepare-quality-review`
- `import-quality-review`
- task/result schemas
- quality report generation

## Allowed files
- `scripts/quality/external_review.py`
- `scripts/deck_master.py`
- `skills/deck-master/schemas/external_quality_review*.schema.json`
- `skills/deck-master/prompts/quality_reviewer.prompt.md`
- `tests/test_external_quality_review.py`

## Requirements
- No LLM provider.
- Support scopes: semantic, visual, evidence, client-readiness.
- Multiple reviewers can coexist.
- P0/P1 imported findings block client export.
- Bad result does not overwrite old report.
- Archive old report on replace.
- Reviewer/source visible in Review Cockpit.

## Tests
- Prepare task for each scope.
- Import valid review.
- Invalid schema rejection.
- P1 external finding blocks export.
- Multiple reviewer reports coexist.
- Archive on replace.

## Done
External quality findings become first-class quality reports.
