# Task C — Narrative Advisory Task / Result Contract

## Goal
Let Codex perform LLM-based narrative reasoning outside Deck Master and write structured narrative advice back.

## Scope
Implement:
- `prepare-narrative-advice`
- `import-narrative-advice`
- `apply-narrative-advice`
- `deck_narrative_advice_task.v1`
- `deck_narrative_advice.v1`

## Allowed files
- `scripts/advisory/narrative.py`
- `scripts/deck_master.py`
- `skills/deck-master/schemas/narrative_advice*.schema.json`
- `skills/deck-master/prompts/narrative_advisor.prompt.md`
- `tests/test_narrative_advice.py`

## Requirements
- No LLM SDK or provider.
- Prepare task from existing artifacts.
- Import validates output JSON.
- Apply supports dry-run and diff.
- Apply may update page_tasks, claim graph gaps, external narrative gate.
- Old values must be preserved in `narrative_advice_diff.json`.
- Write typed events.

## Tests
- Task creation.
- Valid advice import.
- Invalid advice rejection.
- Dry-run creates diff only.
- Apply updates relevant artifacts.
- Bad JSON does not overwrite.

## Done
Codex can read narrative task, write advice, and Deck Master can apply it safely.
