# Task H — Companion Tool UAT Contracts

## Goal
Define and validate contracts for PPT Library, PPT Deck Pro Max, and PPT Master.

## Scope
Implement:
- `validate-ppt-library-result`
- `validate-generation-result`
- `validate-render-result`
- UAT docs.

## Allowed files
- `scripts/contracts/validators.py`
- `scripts/deck_master.py`
- `docs/uat/ppt-library-contract-uat.md`
- `docs/uat/ppt-deck-pro-max-contract-uat.md`
- `docs/uat/ppt-master-contract-uat.md`
- `tests/test_companion_contract_validators.py`

## Requirements
- Validate required fields.
- Distinguish errors and warnings.
- PPT Library result missing screenshot is warning.
- Missing canonical id is warning if fallback possible.
- Completed generation result requires artifact or preview.
- Failed generation result requires errors.
- Render result validates artifact/page count.

## Tests
- Valid PPT Library result.
- Missing required field.
- Missing screenshot warning.
- Valid generation result.
- Invalid generation result.
- Valid render result.

## Done
Deck Master can check companion tool outputs before consuming them.
