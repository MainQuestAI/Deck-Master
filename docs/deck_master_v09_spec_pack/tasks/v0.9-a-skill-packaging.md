# Task A — Skill Packaging & Installation

## Goal
Package Deck Master as a Codex-first Agent skill stored in this repository and symlinked into external Agent skill directories.

## Scope
Implement:
- `skills/deck-master/`
- `install-skill`
- `validate-skill`
- `uninstall-skill`

## Allowed files
- `scripts/skills/installer.py`
- `scripts/deck_master.py`
- `skills/deck-master/**`
- `tests/test_skill_installation.py`

## Requirements
- Use symlink by default.
- Never overwrite a real directory unless explicitly safe.
- `--force` can replace an existing symlink.
- Support `codex`, `claude-code`, `hermes`, `custom`.
- No dependency on any Agent private API.
- CLI outputs JSON.
- Typed events or global install log required.

## Tests
- Install into temp skill dir.
- Reinstall idempotent.
- Validate active symlink.
- Uninstall only symlink.
- Existing real dir blocks install.
- Unsupported target returns clear error.

## Done
Codex can discover `skills/deck-master/SKILL.md` through a symlinked skill directory.
