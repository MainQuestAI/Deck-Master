# Screenshot Checklist

## 1. 截图目标

截图用于三类材料：

1. README 首屏说明。
2. Technical Preview release note。
3. 后续网站或 GitHub 社交预览素材。

## 2. 必须截图

| ID | Viewport | 页面 | 用途 | 验收点 |
|---|---|---|---|---|
| SS-001 | 1440x900 | Review Desk 首屏 | README | 命名、主动作、状态带清楚 |
| SS-002 | 1280x720 | Review Desk 主工作区 | release note | 当前页、右栏动作、底部抽屉不冲突 |
| SS-003 | 390x844 | Review Desk 移动首屏 | QA | 文本不溢出，按钮不重叠 |
| SS-004 | 1440x900 | Demo run 审批前 | README | 公开 fixture 数据可读 |
| SS-005 | 1440x900 | Demo run 审批后 | release note | readiness/export 状态可见 |

## 3. 截图前检查

1. 页面无作者本机路径。
2. 页面无 placeholder。
3. 页面无旧命名“方案项目工作台”。
4. 主动作是琥珀铜。
5. 背景和面板符合冷墨底 + 发丝实面方向。
