# Deck Master Web UI Spec Bundle

本目录承接 Deck Master Review Desk UI 的专项规范，服务 `scripts/preview/static/` 的后续迭代。

当前收口范围：

- 顶部状态带 + 三栏审查台 + 底部抽屉的 Review Desk 结构
- 用户侧统一用语：`Build / Deck Builder`
- planned run / no-preview shell 的空态呈现
- 交付预览、阻断原因、审批与导出状态的一致表达
- 语言切换保留禁用态，等真实双语实现后再开放
