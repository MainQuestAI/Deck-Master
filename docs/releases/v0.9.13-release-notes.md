# Deck Master v0.9.13 Release Notes

v0.9.13 将 Deck Master 收敛为可一次安装的 Agent-facing Product Capability Suite。PPT Master、PPT Library、PPT Deck Pro Max、PPT Quality Gate 作为 Deck Master required product capabilities 随 release tree 分发，并通过 Deck Master run state 回写产物。

## Key Changes

- 新增 `product-capability-manifest.json`，明确 Deck Master required skills 和 product capability skills。
- 新增 `suite-build-release-tree`，生成安装用 release tree。
- 扩展 `suite-install` / `suite-status`，required suite 必须全量 ready。
- 新增 `deck-planner`、`deck-review`、`ppt-master`、`ppt-library`、`ppt-deck-pro-max`、`ppt-quality-gate` Skill packages。
- 新增 PPT Master 最小 render path：`deck-master render --format html --fixture-safe`。
- render result canonical 路径统一为 `render_results/render_result.json`。
- generation discovery 默认优先 bundled `ppt-deck-pro-max` capability，显式 CLI override 保留高级路径。
- `benchmark-rc-report` 对 required render result 增加强阻断。
- Skill 文档更新为 Agent 驱动 setup ceremony，并推荐 `setup --install-suite`。

## Compatibility

- `install-skill` / `validate-skill` / `uninstall-skill` 保持 single skill 兼容。
- legacy render result 路径仍可读取，但 v0.9.13 新写入必须归一到 canonical path。
- workspace / global tool registry 仍可作为 capability fallback 或显式 override 使用。

## Validation

Release 前必须通过：

```bash
python3 -m unittest discover -s tests
git diff --check HEAD
```

并完成 `docs/qa/v0.9.13-product-capability-suite-qa.md` 里的 smoke 证据收集。
