# Deck Master v0.9.12 Release Notes

## Summary

v0.9.12 将 Deck Master 升级为 Skill Suite Runtime。核心变化是：安装与 readiness 进入纯读可诊断状态，PPT Library / PPT Quality Gate / PPT Deck Pro Max 的外部结果必须回写 Deck Master run，Review Cockpit 可以看到 suite、import、quality、feedback 的运行状态。

## New CLI

- `deck-master setup-status --include-suite --output json`
- `deck-master suite-status --output json`
- `deck-master suite-install --target codex`
- `deck-master suite-repair --target codex`
- `deck-master install-skill --suite --target codex`
- `deck-master library-status --output json`
- `deck-master import-library-selection --run-id <run_id> --input <selection.json>`
- `deck-master import-quality-findings --run-id <run_id> --input <findings.json>`
- `deck-master record-library-feedback --run-id <run_id> --page-task-id <page_id> --beat-id <beat_id> --candidate-id <candidate_id> --outcome <value>`

## Runtime Changes

- `suite-status` 与 `setup-status --include-suite` 保持纯读，不写 HOME、workspace、run artifact、events 或 install log。
- Companion skill matrix 覆盖 `deck-master`、`ppt-library`、`ppt-deck-pro-max`、`ppt-quality-gate`，`ppt-master` 为 optional。
- Import log 统一写入 `runs/<run_id>/imports/import_log.jsonl`。
- PPT Library selection 导入写入 canonical 与 legacy 双路径，旧 `decide-sourcing` 继续读取 legacy selection。
- PPT Quality Gate findings 导入后由 Deck Master 重新计算 delivery readiness。
- PPT Deck Pro Max generation result 导入后继续走现有 generation session 状态机。
- `quality_required` 状态继续阻断 export，直到 fresh quality gate 通过。
- Feedback 默认进入 run-local event queue：`external/ppt_library/library_feedback_events.jsonl`。

## Review Cockpit

- `/api/setup-status` 返回 suite readiness。
- `/api/external-results/<run_id>` 返回 runtime readiness 摘要。
- 新增 `/api/runtime-readiness/<run_id>`。
- 外部结果面板显示 suite readiness、import log、blocking quality、pending feedback。

## Compatibility

- 现有 `install-skill / validate-skill / uninstall-skill` 保持单 skill 兼容。
- 现有 `search-library`、`decide-sourcing`、`import-quality-review`、`generation-session import-results` 保持可用。
- 旧 run 没有 import log 时，metrics 和 Cockpit 返回空摘要。
