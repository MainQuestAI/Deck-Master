# Code of Conduct

Deck Master uses a simple contribution standard:

1. Be respectful and direct.
2. Keep technical discussions evidence-based.
3. Do not post private customer data, credentials, or personal information.
4. Do not harass, threaten, or target contributors.
5. Keep issues and pull requests focused on improving the project.

Maintainers may remove comments, close issues, or block participation when behavior damages the project or its users.
